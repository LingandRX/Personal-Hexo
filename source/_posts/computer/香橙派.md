## 温度传感器

> 香橙派有两个温度传感器，一个 cpu 一个 gpu 分别是 zone0 和 zone1

### 查看是什么设备

```shell
cat /sys/class/thermal/thermal_zone0/type cpu_thermal_zone0/type
```
### 查看设备温度

```shell
cat /sys/class/thermal/thermal_zone0/type cpu_thermal_zone0/temp
```
